# 使用指南:

1. 解压缩VerTextBarAnimation.zip

2. 将文件夹放到任何web服务器中,推荐liveserver,http-server或nginx

3. 打开浏览器访问index.html即可(推荐安装了独立显卡的电脑,浏览器推荐chrome)

# 查看:

1. 不跑web服务器的话,可直接查看animation.gif文件预览成果

2. data3.json是地理信息数据
